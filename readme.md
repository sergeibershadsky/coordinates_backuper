## Утилита для сохранения координат общественного транспорта с www.marsruty.ru/krasnodar/ на Я.Диск

### Использование

Отредактируйте файл с переменными окружения


```bash
cp .env.example .env
```


Запуск с помощью docker-compose


```bash
docker-compose build
docker-compose up -d
```
